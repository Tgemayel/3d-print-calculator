3D Print Estimator API

API Endpoint

POST /estimate

Upload an STL file to get estimated build time and material usage.

Request Format

Send a multipart form data request with the following parameter:

  • file: STL file (max size: 16MB)

Response Format

{
    "success": true,
    "estimates": {
        "build_time_minutes": float,
        "material_usage_grams": float,
        "estimated_cost": float
    }
}
